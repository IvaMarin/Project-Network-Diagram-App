Инструкция:
1. Добавить вершину - нажмите СКМ* в любом месте поля
2. Соединить вершины - нажать ПКМ* по двум вершинам
3. Переместить вершину - удерживайте ЛКМ* любую созданную вершину
4. Удалить вершину - наведите курсор мыши на удаляемую вершину и нажмите delete на клавиатуре

*ЛКМ - левая кнопка мыши
*ПКМ - правая кнопка мыши
*СКМ - средняя кнопка мыши
