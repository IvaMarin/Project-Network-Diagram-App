Чтобы запустить программу на 16 тестах, выполните в терминале следующие команды:

1) make clean
2) make 
3) make run

Тесты формируются в следующем виде:
- на первой стороке написаны два числа: n -- количество вершин и m -- количество ребер
- следующие m строк содержат описания ребер: из какой вершины, в какую вершину, вес ребра
